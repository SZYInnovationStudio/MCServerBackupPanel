# MCServerBackupPanel (MCSBP)

一个专为 MCSManager 设计的服务器备份管理系统，由 SZY创新工作室 开发。

## 功能特点

- 🖥️ **服务器管理** - 支持自动检测 MCSManager 实例和手动添加服务器
- 📦 **备份任务** - 为每个服务器配置独立的备份任务，支持定时备份
- 🔄 **一键恢复** - 从任意备份点快速恢复服务器数据
- 🌐 **公开下载** - 可将备份设置为公开，供玩家下载地图
- 🎨 **现代化界面** - 响应式设计，美观易用
- 🔒 **安全防护** - 管理员登录验证，防止未授权访问

## 系统要求

- PHP 7.3 或更高版本
- MySQL 5.7 或更高版本
- PHP 扩展：PDO、PDO MySQL、ZipArchive、JSON

## 安装步骤

1. 将所有文件上传到网站目录
2. 确保目录具有写入权限
3. 访问 `install.php` 按照向导完成安装
4. 安装完成后 `install.php` 会自动删除

## 使用说明

### 添加服务器

1. **自动检测**：输入 MCSManager Daemon 的安装路径，系统自动扫描实例
2. **手动添加**：直接输入服务器名称和文件目录绝对路径

### 创建备份任务

- 备份源文件夹：要备份的服务器目录
- 备份保存路径：备份文件存放位置
- 备份文件名：支持 `{date}` 和 `{server}` 变量
- 定时执行：设置自动备份时间（可选）

### 管理备份

- 查看服务器的所有备份列表
- 立即执行备份任务
- 从备份恢复服务器数据
- 设置备份为公开供玩家下载

## 安全建议

1. 修改默认的数据库连接信息
2. 设置强密码保护管理员账户
3. 定期清理旧的备份文件
4. 建议配置 HTTPS 访问
